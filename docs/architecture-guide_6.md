# Architecture documentation and diagrams — Part 6

## Overview
POSVault is a STX staking vault with governance token rewards and on-chain proposal voting.

### Section 6 Topics
#### Governance System
Token holders create proposals, vote with token-weighted ballots (1008-block window),
and execute passed proposals to modify vault parameters.
